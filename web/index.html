DOOM HTML

Canvas launcher for the Doom engine with local IWAD upload.

1. Select an IWAD

Pick a local .wad file before starting. The file is stored in your browser for this session only.

No IWAD selected.

2. Start engine

The engine bootstrap is prepared here. Hook your Emscripten build output into doom-loader.js.

Controls

  • Arrow keys / WASD – move
  • Ctrl / mouse – fire
  • Space – use
  • Shift – run
